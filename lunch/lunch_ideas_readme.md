A simple Streamlit app that suggests a random lunch idea from 100 Singapore / Chinatown food items.

Click the **"Let's lunch!"** button to get a random lunch menu suggestion.

## Features
- Korean subtitle for fun
- Styled button and lunch display
- 100 curated Singapore lunch ideas

## How to Run
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/lunch-ideas.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run lunch_ideas_streamlit.py
   ```
